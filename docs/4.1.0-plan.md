# UMM 4.1.0 版本计划

## 一、设计原则（必须遵守）

### 1.1 核心约束

**避免过度设计**：
- ❌ 避免大型单文件（单文件不超过300行）
- ❌ 避免层级过深（目录嵌套不超过3层）
- ❌ 避免为DDD而DDD（简单问题简单解决）

**优先DDD和子模块**：
- ✅ 优先考虑领域驱动设计（DDD）
- ✅ 优先考虑子模块化
- ✅ 但要平衡模块化和可维护性

**样式系统约束**：
- ✅ 设计系统必须适配Tailwind CSS v4
- ✅ 使用`umm`前缀避免样式冲突（如`umm-p-2`、`umm-text-center`）
- ✅ 统一的设计令牌源

### 1.2 平衡原则

```
简单问题 → 简单解决（不强制DDD）
复杂问题 → DDD + 子模块（但层级不超过3层）
样式问题 → Tailwind v4 + umm前缀
逻辑问题 → composables + shared模块
```

### 1.3 响应式断点系统（13级）

```css
/* 完整的断点系统，覆盖所有屏幕尺寸 */
```

### 11.1 内存管理优化

**统一内存管理**：
```typescript
// src/shared/memory/index.ts
export class MemoryManager {
  private static instance: MemoryManager
  private observers: Map<string, MutationObserver>
  private listeners: Map<string, EventListener>
  private timers: Map<string, number>
  
  private constructor() {
    this.observers = new Map()
    this.listeners = new Map()
    this.timers = new Map()
  }
  
  static getInstance(): MemoryManager {
    if (!MemoryManager.instance) {
      MemoryManager.instance = new MemoryManager()
    }
    return MemoryManager.instance
  }
  
  // 注册MutationObserver
  registerObserver(key: string, observer: MutationObserver): void {
    this.observers.set(key, observer)
  }
  
  // 注册事件监听器
  registerListener(key: string, element: Element, event: string, handler: Function): void {
    element.addEventListener(event, handler as EventListener)
    this.listeners.set(key, { element, event, handler } as any)
  }
  
  // 注册定时器
  registerTimer(key: string, timer: number): void {
    this.timers.set(key, timer)
  }
  
  // 清理所有资源
  cleanup(): void {
    // 清理MutationObserver
    for (const observer of this.observers.values()) {
      observer.disconnect()
    }
    this.observers.clear()
    
    // 清理事件监听器
    for (const [key, listener] of this.listeners.entries()) {
      listener.element.removeEventListener(listener.event, listener.handler as EventListener)
    }
    this.listeners.clear()
    
    // 清理定时器
    for (const timer of this.timers.values()) {
      clearTimeout(timer)
      clearInterval(timer)
    }
    this.timers.clear()
  }
  
  // 清理指定资源
  cleanupByKey(key: string): void {
    // 清理MutationObserver
    const observer = this.observers.get(key)
    if (observer) {
      observer.disconnect()
      this.observers.delete(key)
    }
    
    // 清理事件监听器
    const listener = this.listeners.get(key)
    if (listener) {
      listener.element.removeEventListener(listener.event, listener.handler as EventListener)
      this.listeners.delete(key)
    }
    
    // 清理定时器
    const timer = this.timers.get(key)
    if (timer) {
      clearTimeout(timer)
      clearInterval(timer)
      this.timers.delete(key)
    }
  }
}
```

**内存管理规范**：
- 所有MutationObserver必须注册
- 所有事件监听器必须注册
- 所有定时器必须注册
- 页面卸载时必须清理

### 11.2 缓存策略优化

**优化缓存策略**：
```typescript
// src/shared/cache/index.ts
export class CacheManager {
  private static instance: CacheManager
  private cache: Map<string, CacheEntry>
  private maxSize: number
  private defaultTTL: number
  
  private constructor() {
    this.cache = new Map()
    this.maxSize = 1000
    this.defaultTTL = 5 * 60 * 1000 // 5分钟
  }
  
  static getInstance(): CacheManager {
    if (!CacheManager.instance) {
      CacheManager.instance = new CacheManager()
    }
    return CacheManager.instance
  }
  
  // 获取缓存
  get<T>(key: string): T | null {
    const entry = this.cache.get(key)
    if (!entry) return null
    
    if (entry.isExpired()) {
      this.cache.delete(key)
      return null
    }
    
    return entry.data as T
  }
  
  // 设置缓存
  set<T>(key: string, data: T, ttl?: number): void {
    // 检查缓存大小
    if (this.cache.size >= this.maxSize) {
      this.evictLRU()
    }
    
    const entry: CacheEntry = {
      data,
      timestamp: Date.now(),
      ttl: ttl || this.defaultTTL,
      lastAccess: Date.now(),
    }
    
    this.cache.set(key, entry)
  }
  
  // 删除缓存
  delete(key: string): void {
    this.cache.delete(key)
  }
  
  // 清空缓存
  clear(): void {
    this.cache.clear()
  }
  
  // LRU淘汰
  private evictLRU(): void {
    let oldestKey: string | null = null
    let oldestTime = Infinity
    
    for (const [key, entry] of this.cache.entries()) {
      if (entry.lastAccess < oldestTime) {
        oldestTime = entry.lastAccess
        oldestKey = key
      }
    }
    
    if (oldestKey) {
      this.cache.delete(oldestKey)
    }
  }
  
  // 清理过期缓存
  cleanup(): void {
    for (const [key, entry] of this.cache.entries()) {
      if (entry.isExpired()) {
        this.cache.delete(key)
      }
    }
  }
}
```

**缓存策略规范**：
- 缓存大小有上限
- 支持LRU淘汰
- 支持TTL过期
- 定期清理过期缓存

### 11.3 减少重复计算

**优化重复计算**：
```typescript
// src/shared/memo/index.ts
export class MemoManager {
  private static instance: MemoManager
  private cache: Map<string, any>
  
  private constructor() {
    this.cache = new Map()
  }
  
  static getInstance(): MemoManager {
    if (!MemoManager.instance) {
      MemoManager.instance = new MemoManager()
    }
    return MemoManager.instance
  }
  
  // 记忆化函数
  memoize<T>(key: string, fn: () => T): T {
    if (this.cache.has(key)) {
      return this.cache.get(key) as T
    }
    
    const result = fn()
    this.cache.set(key, result)
    return result
  }
  
  // 异步记忆化函数
  async memoizeAsync<T>(key: string, fn: () => Promise<T>): Promise<T> {
    if (this.cache.has(key)) {
      return this.cache.get(key) as T
    }
    
    const result = await fn()
    this.cache.set(key, result)
    return result
  }
  
  // 清除记忆
  clear(key?: string): void {
    if (key) {
      this.cache.delete(key)
    } else {
      this.cache.clear()
    }
  }
}
```

**重复计算规范**：
- 计算结果必须缓存
- 缓存必须有失效机制
- 定期清理无用缓存
- 避免重复DOM查询

```typescript
// src/shared/security/xss.ts
export class XSSProtection {
  // HTML编码
  static encodeHTML(text: string): string {
    const map: Record<string, string> = {
      '&': '&amp;',
      '<': '&lt;',
      '>': '&gt;',
      '"': '&quot;',
      "'": '&#039;',
    }
    return text.replace(/[&<>"']/g, (m) => map[m])
  }
  
  // JavaScript编码
  static encodeJS(text: string): string {
    return text.replace(/[^\w\s]/g, (m) => `\\${m.charCodeAt(0).toString(16)}`)
  }
  
  // URL编码
  static encodeURL(text: string): string {
    return encodeURIComponent(text)
  }
  
  // 清理用户输入
  static sanitize(input: string): string {
    return this.encodeHTML(input)
  }
  
  // 验证输入
  static validate(input: string, pattern: RegExp): boolean {
    return pattern.test(input)
  }
}
```

```typescript
// src/shared/audit/index.ts
export class AuditLogger {
  private static instance: AuditLogger
  private logs: AuditLog[]
  
  private constructor() {
    this.logs = []
  }
  
  static getInstance(): AuditLogger {
    if (!AuditLogger.instance) {
      AuditLogger.instance = new AuditLogger()
    }
    return AuditLogger.instance
  }
  
  // 记录审计日志
  log(action: string, details: any): void {
    const entry: AuditLog = {
      timestamp: new Date().toISOString(),
      action,
      details,
      userAgent: navigator.userAgent,
    }
    
    this.logs.push(entry)
    
    // 持久化到IndexedDB
    this.persist(entry)
  }
  
  // 持久化到IndexedDB
  private async persist(entry: AuditLog): Promise<void> {
    try {
      await chrome.runtime.sendMessage({
        type: 'AUDIT_LOG',
        payload: entry,
      })
    } catch (error) {
      console.error('[AuditLogger]', error)
    }
  }
  
  // 获取审计日志
  getLogs(filter?: AuditFilter): AuditLog[] {
    if (!filter) return this.logs
    
    return this.logs.filter((log) => {
      if (filter.action && log.action !== filter.action) return false
      if (filter.startTime && new Date(log.timestamp) < filter.startTime) return false
      if (filter.endTime && new Date(log.timestamp) > filter.endTime) return false
      return true
    })
  }
}

// 审计日志接口
interface AuditLog {
  timestamp: string
  action: string
  details: any
  userAgent: string
}

// 审计过滤器接口
interface AuditFilter {
  action?: string
  startTime?: Date
  endTime?: Date
}
```
